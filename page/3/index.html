试用GOOGLE TTS朗读

发表于 2019-07-30

今天看到篇关于听书的博文（https://www.playpcesor.com/2019/07/android-talkback.html），介绍了使用原生GOOGLE TTS朗读文字。一直用讯飞语音十多年了，期间讯飞语音只更新过几次版本，而且朗读效果提升很小，GOOGLE TTS读中文上一次试用至少是6年前的事了，当时感觉比讯飞语音差太多。于是就马上试用了一下当前的GOOGLE TTS，看看效果，结果GOOGLE TTS听感上果然超越了讯飞语音，自然流畅。顺手搜索了下目前主流三大中文语音引擎评价，最好的是百度，其次GOOGLE，最次讯飞。俺是用语音听电子书的，原来用的多看并不支持GOOGLE TTS。搜索一番静读天下有，安装之没有朗读选项，继续找专业版，安装终于出现了朗读选项。

9-17:在使用过一个月后，已经习惯了使用GOOGLE TTS，优点是：听感稍微好一点，GOOGLE TTS在识别长文字段不同语义间断开片刻，这点做得很棒，听了这么久，没见到断错过，而且相信GOOGLE升级的频率；后省电大为优化，比之前使用讯飞语音+多看的方式省电50%以上（估计这二货在后台做了不少见不得人的事）。缺点：在关闭静读天下及断开GOOGLE网络连接后，重新打开静读天下会导致无法朗读，应该是每次连接GOOGLE TTS会连接一次网络。解决很简单，不关闭静读天下即可。

9-25:更正，在关闭静读天下或后台一段时间，GOOGLE TTS将无法朗读，此时需要重新连接一下网络即可。我是在备用手机上使用的，很少联网，故经常遇到这个问题。

漫画之神手冢治虫

发表于 2019-07-12

近日看《多罗罗》评价时，一老人提到日本漫画之神是手冢治虫。我一直以为漫画第一把交椅是宫崎骏，手冢治虫印象中只记得其几部好老的动画《阿童木》、《三眼童子》、《怪医黑杰克》，显得比较低龄科幻化。

于是我开始补看手冢治虫代表作《火鸟》，顿时惊为天人，一共12章，每章的时代背景及故事完全不同却又有微妙的联系，立意深远，每章都能独立拍成一部动画或者电影大作。可惜的是从54年开始连载一直到作者逝世还有最后一篇《现代篇》未启笔，作者是打算用最后一篇统一联系所有章节。 比如1967年《未来篇》讲述的人类末世靠人工智能来维持生存的故事背景，更是直接被宫崎骏代表作82年开始连载《风之谷》借用。要知道60年代连电脑都没有，世面上仅仅只有黑白电视、收录机。

动画版虽老，但配曲非常优秀，不管是看动画还是漫画，都喜欢配上”望郷”、”PHOENIX - ベートーベン悲怆より - メイン?テーマ”，不知不觉间已是泪流满面。神之作，每每看完一集，每每看完一遍，都会感觉视野的开拓，精神的升华

手贱换山地车配件

发表于 2019-07-08

目前的挑战者从13年9月入手，已经骑了快6年。上个月起就发现14T飞轮+32T中盘会跳链，检查了下是14T飞轮和32T中盘磨损严重了，链条更是磨损，平时习惯于用这二个齿在跑。全部换太贵，光是一个一体牙盘就得300-500，网上查了可以单独换片，于是计划：换了个新链条，换14T飞轮片，32T牙盘中盘也单独换。

东西淘宝买齐花了170+ 牙盘少个曲柄盖拆卸工具，手贱只换了个链条，结果只剩下1、3牙盘和1-3、 8-9速飞轮正常，其它全部跳链。飞轮死活拎不下来，跑了附近二家修电单车店，老板也没办法，借用工具同样拎不开。没办法了只好去上班路上一家骑行车店找老板换新飞轮+牙盘修补片=180。全套下来一下350没了，老板说中轴也不行了，换的话要200，我说算了，能骑一年是一年吧

结论是：根本不需要换，之前也就一个档不行，可以换其它档骑，一共3*9=27个档位，至少撑1年绝对没问题。等飞轮和牙盘进一步磨损再换好点的配件也不迟。这180只换到一个淘宝上的山寨50块飞轮，非常的不值。以这样的质量加上我天天骑的强度，估计只能撑2-3年

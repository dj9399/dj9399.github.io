头大的HEXO报错

发表于 2018-10-31

github BLOG提交报错十来天了，报错如下
Error: packet_write_wait: Connection to xxx.xxx.xxx.xxx port 22: Broken pipe
网上找来一大堆的解决方法无效，最后感觉是公司防火墙的原因，于是换手机热点，重新提交,OK成功！
下次遇上这种问题：先判断是否是SSH KEY的问题，测试方法： ssh -T git@github.com、 ssh -T git@git.coding.net
然后下一步就可能是网络或端口被QIANG了，这就只能换个网络来测试了

解决MTP USB连接电脑文件或文件夹不同步的问题

发表于 2018-10-31

用手机缓存1080视频文件，然后到电脑上拷贝，却看不到文件
因为MTP机制的设定，当机器内存储新增或变化了文件或文件夹，会有一个同步时间，如果在机器内部索引没有自动更新前，就插入电脑用MTP的方式访问手机内文件的话，会发现看到的文件与手机实际文件不一致的情况。

解决方法：
试过重启，无效。一般要等上半天，手机才会更新缓存
安装media scanner 或者 sd scanner，任选其一安装之。
每次发现存在这个情况，就用这两应用的手动扫描功能执行一遍，即可

身份证到期补办流程

发表于 2018-10-23

身份证到期，查了补办要以下资料
1、申请人原申领的居民身份证
2、深圳市《居民身份证数字相片采集检测回执》
http://www.rzzx.com.cn
3、原居民身份证复印件一份
预约入口（http://bsy.sz.bendibao.com/bsyDetail/607655.html）：其实就是深圳J C局了
其中相片是最难搞定的，用小米5不行然后用小米8照了N多相片，加上自己扣图和PS，浪费了几个小时，仍然不合要求。自己摸索出的要求如下：358x441像素文件小于28K or 540x665像素文件小于100K，白底，脸不能反光，不能有阴影，衣服要有领不能穿T裇，真TM多要求也没有相关说明。最后还是去照相馆，馆长单反+闪光 一发入魂。

使用you-get下载优酷视频

发表于 2018-06-20

需求：准备下点优酷DOTA2比赛视频端午在家观看，打算用youtube-dl齐名的you-get来进行
环境：要求python3+pip环境
安装you-get
pip install you-get
下载视频
you-get -i url查看网页视频的信息。例如，我要下载优酷网小熊flippy的视频，就输入类似下面的命令。
you-get -i http://v.youku.com/v_show/id_XMjY3NzY3NTMyNA==.html?spm=a2h0z.8244218.2371631.3&from=y1.9-3.1
you-get会分析视频的信息，然后将可下载的视频按照清晰度列出来。
site: 优酷 (Youku)
title: 猎奇神作《2Dark》05丨地狱斗狗场的幕后真凶！
streams: # Available quality and codecs
​ [ DEFAULT ] _

- format:        hd2
  container:     flv
  video-profile: 超清
  size:          455.1 MiB (477181761 bytes)
# download-with: you-get --format=hd2 [URL]

- format:        mp4
  container:     mp4
  video-profile: 高清
  size:          242.6 MiB (254337261 bytes)
# download-with: you-get --format=mp4 [URL]

- format:        flvhd
  container:     flv
  video-profile: 标清
  size:          117.5 MiB (123191208 bytes)
# download-with: you-get --format=flvhd [URL]

例如我要下载最清晰的版本，就输入超清那里提示的命令。
you-get –format=hd2 http://v.youku.com/v_show/id_XMjY3NzY3NTMyNA==.html?spm=a2h0z.8244218.2371631.3&from=y1.9-3.1
然后下载就会开始，稍等片刻就会下载完成并自动将多个视频分片合并

支持下载列表，比如优酷个人空间里的所有视频,播放列表里的所有视频等等
指定下载目录：you-get -o E:/Desktop https://www.youtube.com/watch?v=vnkhgZPLys4
下载多个url视频可以用 you-get –format=mp4hd url1 url2 url3(中间用空格隔开即可)
很强大，很好用。可惜you-get不能修改默认画质，默认画质为最高，而我基本只要mp4hd(高清)即可

树莓派模拟器折腾记

发表于 2019-10-23

年初想把一张SD卡安装一个模拟器。首先看老外的评价，普遍是LAKKA比较好用，而且LAKKA可以使用NOOBS安装。于是采用NOOBS LITE，单独安装LAKKA试用。因为小，所以很快就安装好了，然后打开LAKKA的SAMBA，从网络拷贝了几个ROM过去，发现扫不出来（扫后没出现任何游戏），LAKKA的资料基本都英文，就没继续折腾了。最后图方便从网上下个RetroPie16G的整合包，设置很是豪华，光是开机画面就是街机开机动画，内置了不少NEOGEO和CAPCOM游戏，挺不错。
到了9月，添加了新ROM，主要是重玩MD光之继承者，顺便就升级最新软件。升级后，之前内置的游戏就突然玩不了了，还出现大堆问题。不过光之继承者可以玩，也就没继续折腾
10月国庆把光之继承者玩通关了。于是继续抽时间来折腾，这次打算安装官方retropie 安装过程很简单，主要是官方源太慢（GITHUB被强限速的原因），最后百毒盘搜索到4.4版本，官方最新是4.5

记录基本配置过程：1.retropie congfig里选第一个basic install，半小时。2.PACKAGE-安装-drastic（运行NDS），安装lr-mame2003-plus(运行部分MAME)，共10分钟；3.UPDATE ALL INSTALLED PACKAGES升级到最新版本，3-5小时 4.显示中文ROM：sudo apt-get install fonts-droid-fallback 5.开启SSH，默认用户密码pi/raspberry最好改下 6.插网线访问：\Retropie(或IP)，传ROM即可，3B的2G无线网会传到想哭 7.RetroArch转到XMB界面（RGUI为默认的土界面），setting–driver–Menu Driver改为XMB，configuretins中save。8.RetroArch切换中文后乱码，换系统核心字库/usr/share/retroarch-assets/xmb/monochrome/font.ttf，换成微软雅黑的font就好了

感想：之所以折腾这么久，是因为现在还玩模拟器的是小众，国外玩的有一大帮，资料也是英文居多。说实话模拟器更多是记录了儿时的快乐，有机会时带上同学，或者是小孩子一起来个2P 3P 4P，更是爽上加爽，小时候最好玩的时光，无非是四人班一起旷课打铁钩船长。而且有许多经典游戏如三国5、恶魔城月下夜想曲等等，用手柄加树莓派的感觉更接近于当年用游戏主机玩的感觉，比用电脑+键盘玩要舒服得多

需要在retropie congfig里选第一个basic install，这是会安装所有lr开头的模拟器的必备文件，装完就可以打开了。Retropie镜像中提供的MAME模拟器仅支持0.37b5和0.78两个版本的ROMS，所以有大部分的街机游戏无法正常运行，所以强烈推荐玩家安装lr-mame2003-plus，然后使用这个模拟器版本进行游戏，此版本支持0.78-0.188的绝大部分ROMS，目前尚未发现无法运行的街机游戏。
最坑爹的地方就是进游戏无法呼出菜单，需要按以下组合快捷键。如果不进行设置就只能用默认的游戏键盘，且只能单人游戏。还好我的北通50块手柄上一通乱按给找到了，左上按钮加上SELECT退出菜单，左上按钮加上X呼出菜单
RetroPie的一些內建快捷鍵操作方式：
快捷鍵(Hotkeys) 功能(Action)
選擇(Select)+開始(Start) 離開(Exit)
選擇(Select)+右上方按鈕(Right Shoulder) 存檔(Save)
選擇(Select)+左上方按鈕(Left Shoulder) 讀檔(Load)
選擇(Select)+向右按鈕(Right) 增加存檔編號(Input State Slot Increase)
選擇(Select)+向左按鈕(Left) 減少存檔編號(Input State Slot Decrease)
選擇(Select)+X按鈕 RGUI 選單(Menu)
選擇(Select)+B按鈕 重置(Reset)

/opt/retropie/configs/all/emulationstation目录下的downloaded_images是默认封面目录，gamelists目录是游戏列表目录
其中包含若干的文件夹，分别对应roms中的同名目录，注意对号入座
玩nes磁碟游戏恐怕要用nestopia，我记得系统里默认在用fceumm ，你得自己装一个。其实Retripie的wiki说明里也是推荐这个FC模拟器，说它表现很精确，但实际默认又并没有安装。
运行retropie setup —–manage packages —–manage optional packages —–lr-nestopia

How To Get Mame Working On Retropie
放置ROM在rom\fba下，不用解压，重启，然后在游戏界面选择Final Burn Alpha，就可以找到
MD模拟：Retropie中默认安装了 lr-genesis-plus-gx 和 lr-picodrive 两个模拟核心包，加载 ROM 后默认使用 lr-picodrive 运行，但不知什么原因 lr-picodrive 不支持六建手柄（即XYZ无法使用），所以需要在【Retropie Setup】->【Manage packages】->【Manage main packages】中删除 lr-picodrive。
Enable SSH :
Got to Raspi-config in the Retropie configuration menu.
Then click on “Interfacing options” and finally on “SSH”
Answer “Yes” to enable the SSH server.
You can now connect to your Raspberry Pi using SSH.
Default login is pi, default password is raspberry.
设置4人：Select+X to access RGUI.Quick Menu -> Controls Change “User 2 Device Type” to Multitap.Select “Save Game Remap File”
特别需要注意的是，有些游戏（如名将）则需要对rom框体进行调整才能实现4人同时游戏，比如在WinKawaks中调整“Dip开关”，在Mame中调整“机台设置”选项 CPS是进入BIOS设置等等…
淘宝上20块钱左右的山寨PS3蓝牙手柄，可以完美兼容这类系统
PS4手柄，直接用USB线连接即可使用。手感很好。好果使用蓝牙，菜单中的常规蓝牙配对可能有效，或者您可能需要使用所调用的用户空间控制器驱动程序ds4drv。要将控制器置于配对模式，请按住“ 共享”按钮，然后按住PS按钮。几秒钟后，灯条会快速而明亮地闪烁。控制器现在处于配对模式。（https://github.com/RetroPie/RetroPie-Setup/wiki/PS4-Controller）
如果闲置，控制器将不会自行休眠，要强制控制器进入睡眠状态，请按住PS按钮10秒钟。

发表于 2017-08-08

需求：购物水评太多，一页页翻看实在是累。于是只看追评，还要求过滤字太少和当天的追评。于是慢慢写了二个小爬代码，过滤掉少于10个字的评价，并保存到EXCEL表格，以便查看。还别说双了11就用上了

爬淘宝天猫

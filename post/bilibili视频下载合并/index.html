最近关注巫师3，于是打算下载1080P的视频回家用电脑或者投影看看。找了遍网络，基本就只有bilibili有并可以方便的用手机下载（基本上是满速下载，B站良心唉）。可下载下的视频是分割的，如一个文件分割成了0.blv，1.blv。于是网上找到最方便的合并方法：用ffmpeg合并
1.安装ffmpeg：
教程:
2.在视频文件同目录下新建一个playlist.txt
# playlist.txt与下面文件同目录，使用相对路径
file ‘0.blv’
file ‘1.blv’
3.同目录下CMD窗口运行：ffmpeg -f concat -i filelist.txt -c copy output.flv

output.flv即为合并后的文件，OK EASY

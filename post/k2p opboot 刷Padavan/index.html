k2p opboot 刷Padavan

发表于 2017-11-02

撸了台K2P，准备备用。然而官方系统是不敢用的，于是查下了目前最好的是Padavan，于是开刷吧

参考：

不过新版本刷机有点差别，作了点更改，因此记录一下

准备工作

官改固件
http://www.right.com.cn/forum/thread-217181-1-1.html

通过后台刷入官改固件，里面集成了opboot

备份
先在官改版的后台备份整个编程器固件，此固件是你最后的救命稻草

刷机

K2P padavan公测版
http://www.right.com.cn/forum/thread-218704-1-1.html

下载地址
http://files.80x86.io/router/rom/K2P/

进入Breed ：按复位，再通电，按10秒 ，再放开复位，然后浏览器访问192.168.1.1/index.html，進入固件更新，选无灯的固件，点更新，大约5分钟左右即可完成无灯的固件刷完是192.168.6.1 admin admin

体验

padavan哪里好？从固件管理页面来说，padavan非常的流畅，远超openwrt类，没有丝毫卡顿，点击即达

通过对无线速度，覆盖范围测试，跟官方固件已经没有任何区别

如果DNS 不正常（表现为：打开常用网站如 taobao , tmall ,jd 等非常卡，升级之前正常），请执行一下双清。

集成 中文支持 自动更新 adblock ngrok shellinabox kcptun SQM-QOS

第三方测试

斐讯K2P刷Padavan与原厂固件性能对比测试

pokemon Go fq+gps模拟

发表于 2016-07-28   |  

8月3号更新： 走上了黑科技…没办法啊，用VPN+模拟位置没办法走动，天天都守着几个PO，升级太难太难了。教程：https://high-minded.net/threads/pokemon-go-c-bot-safer-better.50731/﻿

8月4号更新：黑科技使用3小时后，第二天一早就无法登录了，查看日志已经升级到15级。打开pokemon Go直接报错…看了下新闻，老任做了更新，新版本已经使用新API，所有第三方辅助都无法使用。估计是新版本同时禁止了模拟位置，造成无法登录…没办法，过几天再看看吧

按照网上的教程搞定：

1、把坐标复制到模拟位置中的Google play服务的模拟中
2、这时再打开谷歌地图就已经定位到之前选的位置了。
3、完成之后同时打开谷歌地图和游戏，进入游戏你会发现你的位置就是之前定的位置，不同时打开的话会提示红底gps。

需要注意的是你开的VPN所在地区，一定要和模拟位置一致。比如我用的香港VPN就一定要把位置修改到香港。否则的话会被锁

比较麻烦的一点是，只能通过修改模拟位置来移动，因为GPS被修改了，不过配合上https://pokevision.com 这个神器。

搜索了一下：因为是模拟定位，你需要到gps信号强的位置去再一次走动。只能周末去户外尝试一下。

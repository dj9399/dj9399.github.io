入小米5及使用电信卡

发表于 2018-06-20

背景：

乘着618众人大换手机之际，入手个小米5，因为手小，只能用5寸屏小手机，又要求能刷原生系统，加上全网通双卡，基本上只有小米系列。在调查了很久，选择米5，性价比高，性能足够使用(真实原因就是穷)

从数字尾巴以550入手64G黑色版本，85成新，是好价。加上60块买了张半年电信每月20G卡配合一波使用同时代替宽带
收到第一时间在拨号界面输入##64663##进行综合测试，都正常
第二时间安装安安兔进行测试，也正常

测试问题:

测试二天后，发现看视频超过10分钟，温度直接到40多度，超过30分钟，温度直接到50度，导致电信卡断流。查资料后用温控拜拜解决，温度也就升高了几度的样子
发热根本原因还是设计原因：
http://www.miui.com/forum.php?mod=viewthread&tid=8192818
采用锁核心方法：
镧工具箱,火龙820耗电太恶心了，于是乎。。。root安装黑域锁后台。。。。分辨率改成1080p,加上锁了两个大核，综合之后感觉续航还可以
http://tieba.baidu.com/p/5476050629
http://www.miui.com/thread-13948257-1-1.html
论坛反应lineagev15.1 温控调度保守 刷完自己装了镧工具 实话不修改也很流畅 续航亮屏6小时左右 不过既然装了 就试试可以改进不 为了安全温控没改 只是使用了自带大核调用模式 出乎意料 非常流畅 也不热 续航增加明显
http://www.miui.com/forum.php?mod=viewthread&tid=12088324&highlight=Lineage

结论：试锁一个核，无论是大核还是小核，看视频都会一卡一卡，CPU使用率全部上升到将近100%，温度也只是下降了几度的样子，不好用。只能忍了温度高这个缺点，尽量少看视频吧

可刷系统调查：

刷了各种类原生系统的，表示已经折腾够了，已经转回 miui9 了，什么都不用折腾，还省电
Lineage+OpenGapp(aroma) 安装 opengapp 时把能卸载的 aosp 的东西都选上，然后你就得到了一个完美的系统
lineage 15.1极致续航
手持 mi5 + lineageos，续航一天一充，自带相机确实渣，但是有这么一个神器可以用上 pixel 的黑科技： https://www.xda-developers.com/google-camera-hdr-ported/
我用的小米 5，刷了 LineageOS 感觉稳定到爆啊，现在每周三我就自己升级下，从来（我说的是从来）没碰到过什么应用崩溃，也可能我的应用比较少，大约是 90 来个
我以前一直用原生系统，去年买了小米才开始用 MIUI。MIUI 虽然有很多方便的功能，但是 Big View 通知、通知栏下拉瓷贴接口、双击多任务返回上一个程序这些原生系统就有的的功能都加进去。 特别是 Big View 通知，android5.1 就有了，现在 MIUI 里还不好用，可能是不打算加进去了。 总的算起来，刷原生要牺牲 mipay，mipush，公交卡等功能，我觉得还是可以接受的
国际版，开发版，无广告，已root，带xp框架，带谷歌全家桶，除了多了个谷歌全家桶，不需要自己装谷歌套件，别的方面感觉并没有优势，没广告是假的，现在国际版也是有广告的，而且看上去比国内版还恶心。国际版不如原生
米5刷的Los14.1，可以装xposed框架之后使用指纹支付这个软件，然后可以实现微信，支付宝，淘宝以及QQ的指纹支付，挺好用的
小米原生识别电信卡完美解决方案
由于个人需要想在MIUI国际版上使用小米公交卡功能，希望能够给出尽量详细的教程
https://pan.baidu.com/s/1c2B3qVY，密码：khdy，把两个APP下载然后安装，你就会发现楼上一群人都不懂在那瞎说。我亲测，国际版MIUI9.8.31开发版。

结论：还是继续使用miui9开发版，不需要折磨，好处是miui系统足够完善，而且官方系统最稳定，可以继续升级。

后续：

6.27
经过4天测试，基本满足个人需求，今天已经将主要软件和微信消息进行了迁移，开始以小米5为主力机。NEXUS5正好放家里看视频+google机备用

7.16

已经用了半个月，小米来电拦截做得真不错，真是很有用。但是发现一个最大的问题就是用电信卡发热大，而且在信号稍微差点的地方就会经常断流。网上查了是小米5通病，无解

12.01

到月中就用满了半年，配合异地打折电信流量卡使用很舒服（PING很稳定），之前视频30分钟温度过高导致电信卡断流不知不觉没了（不知道是不是升级了MIUI10的关系，最高拿到手时用的是MIUI9，这点有必要记录一下）。现在再也用不着广东坑爹电信宽带。已经把米5升级到最新MIUI10，然后发现，MIUI10真不如9，有轻微的卡顿，系统广告满天飞(9基本可以全关)，网上有提到小米5MIUI最好用的版本是稳定版8.1.6，不想折磨了，等以后有需要再刷吧。

12.21

半年电信卡到期了，于是购入了一张新卡（北京电信校园卡#12.5元/月，20G/月流量）。发现小米5插入新卡死活不认4G，网上查找各种方法试了无效，找电信客服打了3次电话刷新网络无效，最后无耐说有人说刷机可以解决，于是就刷新了稳定版8.1.6，果然解决了。新问题来了：稳定版8.1.6微信无法关闭小程序，查询解决方法是只能升级到9.2以上版本。先不管，继续用稳定版8.1.6。

2019

1.28

8.1.6版本+北京电信卡遇上个严重的问题，网络有点卡。远不如之前的广东卡流畅，怀疑是北京卡把网络频段给锁死造成。可惜稳定版没有ROOT（至少很麻烦），加上8.1.6远不如之前用MIUI9开发版流畅，负一屏功能也少了，于是还是刷MIUI9最后一个开发版（至少将来再不济了可以继续在开发版本上刷Lineage）。ROOT后加上网络信号大师解锁，默认是BAND1，试了其它几个，改善不大。只有BAND3流畅一点，至少测速提升到了下载30M（BAND1一直是10-20M），上传被限1M。想起之前的广东卡，下载速度一样，上传可以达到10M，现在知道瓶颈就出在上传上了！
之后到公司又测试了速度，上传又可以达到6M，奇怪了，看来是家里附近的基站限制了上传速度

3.1

经过一个月的使用加上测试，网络信号大师分别锁定BAND1、BAND3，下载提升效果并不明显，可喜的是上传终于不会限1M了，在家远程公司电脑流畅了许多。可以初步判断这张卡之前为电信作了限制

4.11

用这张北京手机卡带电脑上网PING SINA值一直在100以上，以前的广东卡稳定在50以下的。

今天无意中看到个贴子指出:电信仍然漫游回回归属地，唉，难怪之前IP显示是北京，原来要到北京漫游。以后买卡一定要小心买广东本地的。

4.19

搜索了下，深圳电信 4G 自动开通了拜访地漫游业务，目前只有广东、江浙部门城市开通了本项业务，其它城市还得等待啊。另外移动是一直自动开通了本功能，联通也于18年基本所有城市开放了本功能。只有电信不知道要啥时候了，不管先打10000号申诉一波

8.21

网上看贴有人说贵州电信卡，居然开通了拜访地接入。刚才试了一下，我的北京电信卡也居然开通了拜访地接入，泪奔。PING 了下 SINA，之前漫游北京是 100+，公司现在 40-60，家里80-100。

树莓派监控

发表于 2020-01-12

突然想折磨下，于是查了一下，树莓派适用的CSI摄像头最便宜，才20块，系统也有开源的motionEyeOS，看了下已经很成熟了
于是有了以下需求：当不在房间时，对房间进行动作捕捉并拍照，最好能上传邮件或网盘。毕竟春节要来了
淘宝入手OV5647 500万像素 光圈2.9 视角 75度 最高支持1080P 其实画质还不到720，不过便宜够用，加邮费32到手
安装好motionEyeOS系统后，启动提示要有网络，因没有多余的网线，只好手写SD卡中的wpa_supplicant.conf以连接WIFI
默认用户名为“admin”，空白密码，直接就能识别到摄像头，方便
访问WEB很流畅，全英文但不难，花了一小时基本配置好了动作捕捉。上传网盘只支持GOOGLE和DROPBOX，家里路由没带XX，只好放弃。邮件报警配置了SINA和163都测试失败，先放弃，采用最简单方式：照片保存本地SD卡
简单总结：花费32得到一个可以动态监控房间并可升级扩展，还是满不错的，至少比起市面上同类200元上下的产品，要实惠多了。让我再次见识了开源的威力
相关配置记录：
wpa_supplicant.conf：
country=CN
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1

network={
ssid=”你WIFI”
key_mgmt=WPA-PSK
psk=”你WIFI密码”
}

motionEyeOS仍需要网络连接才能正常运行；如果您真的不想/不需要一个，
请设置/data/etc/os.conf 中os_networkless为true，如不能修改，则使用/boot/os.conf设置
Why do I get Read-only file system when trying to edit a file：mount -o remount,rw /etc
experct settings中改为NTP，使用国内NTP服务器：ntp1.aliyun.com
/etc/proftpd.conf
/etc/samba/smb.conf
EDITOR=nano crontab -e
30 19 * /sbin/halt now

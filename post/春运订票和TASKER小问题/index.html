春运订票和TASKER小问题

发表于 2016-12-28

12/27 早上到了公司，到10：30的出票点，试着用12306分流抢下票，马上成功的抢到一张火车票，这个软件不愧是V2EX上最为推荐的一款，而且免费，软件本身也做成了绿色版本。5星评价。如果朋友亲威多的话，买个VIP就能使用自动云打码功能，这样使用更加方便。吐嘲下12306的T出机制，T M D一到高峰出票时把你T出去，然后重新输入那变态的验证码。这么做完全是把普通订票用户给拒之门外，春运最高峰票不用软件，不找黄牛，就用这12306垃圾平台，你是根本没有机会能订到票。

今天早上又抢到一张26晚上的高铁票。昨天已经和公司所有需要订票的同事都安利了本软件，凡是有点耐心的真正想抢票的都能通过12306分流抢到票了。实在是感谢12306分流作者：Cheney.小风，一个免费的软件做到这么好，真是没话说了。12306一个花了近十亿的WEB，连十亿的票都卖不好，被人家一个免费的软件直接秒杀。

一直是使用TASKER来进行团贷网的自动定时签到，这几天发现无法正常运行。之前是使用 input tap来模拟单点，于是手动试了二天，发现团贷网在单点情况下确实无法签到，但是用手指重一点点击是可以签到的。推测其机制应该改为了多点感应触摸“签到”或者是长按“签到”，今天在网上找了下资料，可以使用tasker shell input 模拟长按操作：在shell中输入 input swipe 100 500 100 500 1000即可，即可实现在该点上长按一秒(1000ms)。立马搞定，现已经回复正常。

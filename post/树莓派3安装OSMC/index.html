树莓派3安装OSMC

发表于 2017-03-23

入手的树莓派3准备做成影视中心（因为家中有闲置硬盘而且派省电）。按之前查看的资料，安装OSMC最佳。
基本设置如下按照以下系列教程很轻松就把OSMC安装好，http://www.ncnynl.com/archives/201607/230.html

安装过程需要直接配置成中文（不然的话会没有切换中文的选项），结果成了乱码，继续GOOGLEhttp://www.mz6.net/news/2015-07-29/2780.html搞定

默认ssh帐号和密码都是osmc

  1. 基本配置
    設置 root 賬戶密碼，輸入兩次新密碼即可

    $ sudo passwd root
    啓用 root 賬戶登錄
    $ sudo passwd –unlock root

    $su root

    启用 root ssh登陆

    nano /etc/ssh/sshd_config

    修改 PermitRootLogin without-password

    为 PermitRootLogin yes

  2. 切换软件源
    nano /etc/apt/sources.list

    切换为中山+阿里源了，更新一下
    #apt-get update

    #apt-get upgrade

    osmc系统下上面的命令有时执行不了，可以换成apt upgrade

  3. 安裝常用工具
    apt-get install git apt-transport-https psmisc tree wget curl samba htop unzip zip p7zip zsh -y

  4. 安装支持NTFS格式及自动挂载U盘
    ​

发现的问题：1、RMVB无法播放，只有声音2、所有中文插件基本无法使用 3、自带字幕的影片无法显示字幕

安装好的样子：

#树莓派 #高清投影
入门高清投影之一：器材

树莓派3安装OSMC

发表于 2017-03-23   |  

入手的树莓派3准备做成影视中心（因为家中有闲置硬盘而且派省电）。按之前查看的资料，安装OSMC最佳。
基本设置如下按照以下系列教程很轻松就把OSMC安装好，http://www.ncnynl.com/archives/201607/230.html

安装过程需要直接配置成中文（不然的话会没有切换中文的选项），结果成了乱码，继续GOOGLEhttp://www.mz6.net/news/2015-07-29/2780.html搞定ssh ip,默认帐号和密码都是osmc

安装好系统后进行基本配置
設置 root 賬戶密碼，輸入兩次新密碼即可$ sudo passwd root
啓用 root 賬戶登錄
$ sudo passwd –unlock root

启用 root ssh登陆

vi /etc/ssh/sshd_config

修改 PermitRootLogin without-password

为 PermitRootLogin yes

  • 安裝常用工具

    apt-get install git apt-transport-https psmisc tree wget curl samba htop unzip zip p7zip zsh -y

  • 更新系統軟件源

    apt-get update -y && apt-get upgrade -y

    編輯 /etc/apt/sources.list 替換系統默認的軟件源# vi /etc/apt/sources.list
    deb http://mirrors.aliyun.com/raspbian/raspbian/ wheezy main non-free contribdeb-src http://mirrors.aliyun.com/raspbian/raspbian/ wheezy main non-free contrib
    现在我们已经是阿里源了，更新一下
    更改軟件源之後 請更新系統軟件源# apt-get update -y && apt-get upgrade -y
    切記：請勿隨意斷開電源！ 關閉電源前請執行關機命令 或 重啟命令
    靜待系統關閉完成再拔掉電源連接線。必须换阿里云才找到了 fuse-utils，弄了30分钟的NTFS，结果发现在OSMC中直接可以浏览到新插入的NTFS格式U盘，原因是OSMC中预装了autofs

最后呢，想研究下插件，折腾了半天，发现所有的中文插件都死了。用了不…就死了这条心吧暂时就先上高清播放功能吧，下载功能以后再慢慢给加上。优点：树莓派播放720资源很流畅，1080的还没有，因此没有尝试

发现的问题：1、RMVB无法播放，只有声音2、所有中文插件都过老无法使用，国外插件完好又基本访问不了

一怒之下删python

发表于 2017-11-30

最近折磨机器学习tensorflow，开始在cenos虚拟机下安装，可是各种报错，各种坑。

于是直接在本机WIN7进行安装，可是WIN7一开始安装的是python2.7和python3.6 32位版本。需要环境是python 3.5以上64位版本，于是又是各种报错，各种坑。最后干脆了，直接删除python2.7和python3.6，只安装python3.6 64位，之后各种安装终于一步到位，世界终于安静了。试用了下，发现早前的一些BUG也被解决了，NICE希望早点彻底告别python2吧
